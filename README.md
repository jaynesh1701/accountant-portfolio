# Nikhil Pokar - Strategic Financial Partner Portfolio

A high-end, senior designer-quality portfolio website for an accountant/strategic financial partner.

## Key Features
- **Design**: "Precision Flow" concept with Swiss-style typography (Instrument Serif + Manrope).
- **Hero Section**: Interactive WebGL shader background (Three.js) with a light mode aesthetic (White/Sage/Forest Green).
- **Animations**: GSAP (ScrollTrigger) and Lenis for smooth scrolling.
- **CMS-lite**: Project data is managed via a simple `data.js` file for easy updates.

## Tech Stack
- HTML5, CSS3, JavaScript (Vanilla)
- Three.js (WebGL)
- GSAP (Animations)
- Lenis (Smooth Scroll)

## Setup
1. Clone the repository.
2. Open `index.html` in your browser.

## Deployment
This project is ready for deployment on Netlify or Vercel.
1. Push to GitHub.
2. Connect repository to Netlify/Vercel.
3. Publish directory: `/` (Root).
